# Vibe Coding Olympics 🏆

A hackathon project where AI software engineers compete to build the internet's worst startup ideas.

Users submit a ridiculous prompt (for example, **"Netflix for Cats"** or **"Uber for Ghosts"**), choose a build time, and watch a team of fictional AI agents race to create the application.

The goal is to make software engineering entertaining by presenting it as a live competition.

---

## Vision

The long-term vision is to integrate real AI coding agents (OpenAI Codex) that autonomously plan, implement, test, and deliver applications.

The initial MVP intentionally simulates much of this workflow while establishing the architecture needed for future iterations.

---

## MVP Features

* Landing page
* Prompt input
* Build duration selector
* Animated build dashboard
* Simulated AI agent chat
* Build progress tracker
* Generated app preview
